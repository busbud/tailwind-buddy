# Benchmarks

We compare our library against `cva` and `tailwind-variants`

To run them:

`pnpm install`

inside core folder:
`pnpm build`

come back to benchmark folder and run
`node benchmarks.mjs`

## Test

`pnpm test`
To make sure we have the same outputs between libraries
